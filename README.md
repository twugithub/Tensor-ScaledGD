# Guaranteed Nonconvex Low-Rank Tensor Estimation via Scaled Gradient Descent

Matlab implementation of the Scaled Gradient Descent for low-rank tensor estimation under the t-SVD framework (see the JMLR paper [link](https://jmlr.org/papers/v27/25-0012.html)).
This code has been tested in the following computational environment.

* PC: Windows 10 with Matlab R2025a
